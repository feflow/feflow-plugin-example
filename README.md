# feflow-plugin-example

This is an official demo for feflow plugin. It's a simple calculator.

For example, if you install the plugin, you can input command in terminal like this:

```sh
fef add 1 2 3
# output: 6
fef minus 1 2 3
# output: -4
fef multiply 2 2 4
# output: 16
fef divide 2 2 4
# output: 0.25
```
